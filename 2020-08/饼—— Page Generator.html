饼—— Page Generator

CLI

  • build
    增量生成
  • rebuild
    全量生成
  • add <page identifier>
    添加新页面
  • init
    初始化当前目录

目录结构

/.

/doc                    Markdown文档目录
  - *.md
  - ...

/template               生成模板目录
  - /Default
      - Tindex.html
      - Tpage.html
      - Tcategory.html

/.inf                   信息目录
  - sn                      序列标识
  - /track                  文件改动追踪
      - ...

/dest                   生成页面目录
  - /2020-01                日期
      - xxx.html                文档页
      - ...

  - /data
      - /category               分类数据目录
          - xxx.dirc
          - yyy.dirc
          - ...

      - directory.dirc          目录数据
      - categories.json         类别数据

  - index.html              主页

Details

模板文件使用{{}}进行标识，需要填充的表项：

  • 文章内容
  • 时间
  • ...待扩展
